# Auto Income Hub

シンプルな静的サイトです。\
「自動化による収益化の導線」を素早く試すための雛形として作成しています。

## スマホで確認する方法

同じWi-Fiに接続したスマホからアクセスできるように、PC側で以下を実行します。

```bash
python3 -m http.server 8000 --bind 0.0.0.0
```

その後、PCのローカルIPを確認して、スマホのブラウザで開いてください。

```bash
hostname -I
```

例: `http://192.168.1.20:8000`

## ダウンロード機能

- 画面内の「プランをダウンロード」ボタンから `monetization-plan.txt` を保存できます。
- ダウンロード内容には、その日の自動タスクが含まれます。

## 収益化のポイント

- まずは1テーマに絞って記事や素材を増やす
- アフィリエイトリンクは比較・体験コンテンツの中に自然に配置
- メルマガやLINE等で再訪導線を作る
- 計測（PV/CVR/成約率）を毎週見て改善する

> 注意: 完全放置で安定収益は難しいため、運用改善を前提にしてください。
