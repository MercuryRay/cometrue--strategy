# Auto Income Hub

自動化を活用した収益導線を、素早く検証するための静的サイトテンプレートです。

## 起動方法

```bash
python3 -m http.server 8000 --bind 0.0.0.0
```

スマホで確認する場合は同じWi-Fi内でPCのIPにアクセスします。

```bash
hostname -I
```

例: `http://192.168.1.20:8000`

## 実装済み機能

- レスポンシブなLP（スマホ優先）
- 本日のタスク自動表示
- `monetization-plan.txt` ダウンロード

## 注意

自動で収益が得られる保証はありません。運用改善が前提です。
