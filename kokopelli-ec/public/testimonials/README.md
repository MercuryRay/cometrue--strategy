# testimonials/ — Before/After イメージ

D2C LP の「お客様の体験談」セクションで使用する Before/After ビジュアル素材。

## ファイル一覧

| ファイル            | 用途             | 状態   | サイズ    |
| ------------------- | ---------------- | ------ | --------- |
| `fur-before.png`    | 毛艶テーマ       | Before | 1080x1080 |
| `fur-after.png`     | 毛艶テーマ       | After  | 1080x1080 |
| `water-before.png`  | 水の飲み量テーマ | Before | 1080x1080 |
| `water-after.png`   | 水の飲み量テーマ | After  | 1080x1080 |
| `energy-before.png` | 元気度テーマ     | Before | 1080x1080 |
| `energy-after.png`  | 元気度テーマ     | After  | 1080x1080 |

全ファイル 200KB 以下、RGB PNG。

## 表現ルール（重要）

- **架空の顔写真は使用していない** — 全てイラスト／グラフ／アイコンで表現
- **断定表現を回避** — "ココペリ使用後" ではなく "使い始めて数週間の変化（個人の体験談）" 等を採用
- **具体的数値は使わない** — 水の飲み量チャートは定性ラベル（"少し"／"増えた"／"しっかり" など）。ml等の捏造データは書かない
- Before = グレースケール背景 + モノトーン配色（憂い感）
- After = 暖色グラデ + ビビッド（ココペリ緑 #10A37F をアクセントに使用）

## 生成方法

```bash
python C:/Users/timbe/kokopelli-ec/scripts/gen-testimonial-imagery.py
```

Pillow のみで生成（Pillow 標準フォント: YuGothic Bold / Meiryo Bold / Noto Sans JP）。

## ペア運用

LP 側では各ペアを横並び or スライダーで配置し、必ず Before/After セットで表示する。片方だけの掲出は不可。

## 注意事項

- 体験談として掲載する場合、必ず実際の顧客の同意済みレビュー文言と紐付けること
- 景品表示法・薬機法に配慮し、効能効果を謳う文言は使わない（毛並み・飲水量・元気度は観察された事実ベースで記述）
- 画像の差し替え時は同スクリプトのパラメータ（色・テキスト）を編集して再実行
